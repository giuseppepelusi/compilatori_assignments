# ***Compilatori 2025-2026*** 
***Gueham-Mantolini-Pelusi***

Repo for all the assignments of `Compilatori 2025/2026`.

## Structure
Each step of each assignment will be implemented in its own subfolder, the folder structure of each step will be as follows:

```
└─ Assignment
		└─ Exercise
				└─ build
					└─ libPass.so
					└─ CMakeFiles
					└─ Makefile
					└─ ...
				└─ src
					└─ Pass.cpp
					└─ CMakeList.txt
				└─ test
					└─ test.c
					└─ test.ll
					└─ test.optimized.ll
				└─ README.md
```

Follow the steps in each `Exercise README.md` for optimization testing.
